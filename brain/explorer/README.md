# Brain Explorer

Dark-themed single-page explorer for the qwen-agent-collective shared brain:
the 5 agents, their shared/private namespaces, and a grounded-recall console.

Backed by the Qwen/DashScope spine (OpenAI-compatible). `enable_search` gives
the recall console live web grounding. **Note:** the displayed memory records
are still demo fixtures baked into `index.html` — not yet wired to the live
Postgres/Qdrant brain.

## Run locally

Prereq: Node 18+ (for global `fetch`). Reads the repo-root `.env`
(`DASHSCOPE_API_KEY`, `DASHSCOPE_BASE_URL`, `QWEN_CHAT_MODEL`).

```
npm install
npm start        # http://localhost:3000
```

## Layout
- `index.html` — the entire UI (vanilla JS + D3 + 3d-force-graph from CDN).
- `server.mjs` — Express: serves the UI + `POST /api/search` (grounded recall).

Origin: generated by Google AI Studio, then ported off Gemini onto the Qwen spine.
